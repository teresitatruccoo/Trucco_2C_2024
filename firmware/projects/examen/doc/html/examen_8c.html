La tarea se encarga de encender el buzzer si la medicion de distancia es menor a 3 m. Se enciende por 500ms y se apaga. Si la medicion de distancia es mayor a 3m, se apaga.

◆ Controlar_Leds()

void Controlar_Leds ( void )

Controla los LEDs segun el valor de la medicion de distancia.

Si la medicion de distancia es mayor a 5 metros, solo enciende el LED_1 esta encendido. Si es mayor a 3 metros y menor a 5 metros, los LEDs LED_1 y LED_2 estan encendidos. Si es menor a 3 metros, los tres LEDs estan encendidos. En cada caso, se muestra un mensaje en la UART indicando si debe tener precaucion "Precaución, vehículo cerca" o si esta en peligro "Peligro, vehículo cerca".
